# 字符操作

- 或操作 | 和空格将英文字符转换为小写

```
('a' | ' ') = 'a'
('A' | ' ') = 'a'
```

- 与操作 & 和下划线将英文字符转换为大写

```
('b' & '_') = 'B'
('B' & '_') = 'B'
```

- 异或操作 ^ 和空格进行英文字符大小写互换

```
('d' ^ ' ') = 'D'
('D' ^ ' ') = 'd'
```

# 数字操作

判断两个数是否异号

```
((x ^ y) < 0) == true // x & y 异号
```

消除数字二进制表示中的最后一个1

```
n & (n - 1)
```
